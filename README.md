# libsonare Homepage

Homepage for [libsonare](https://libsonare.libraz.net) - built with VitePress.

## Development

```bash
# Install dependencies
yarn install

# Start dev server
yarn dev

# Build for production
yarn build

# Preview production build
yarn preview

# Copy WASM from ../libsonare/dist
yarn copy:wasm
```
