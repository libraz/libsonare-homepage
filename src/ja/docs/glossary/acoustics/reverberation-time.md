# 残響時間（RT60 と EDT）

**残響時間**は、音が止まったあとに部屋がどれだけ鳴り続けるかを表す代表的な指標です。

音源が止まっても、すでに部屋の中にあるエネルギーは消えません。反射のたびに少しずつ失われながら、壁の間を跳ね返り続け、やがて暗騒音に紛れていきます。そこまでにかかる時間こそが、その部屋の響きです。音響処理されたボーカルブースはほぼ瞬時に減衰し、石造りの大聖堂は何秒も尾を引きます。

## RT60: 60 dB の減衰

**RT60** は、音源が止まってから残響の尾が **60 デシベル**（エネルギーで 100 万分の 1）減衰するまでの時間（秒）です。60 dB という値は、静かなホールで大きな音から可聴限界までのおおよその幅に当たるという歴史的な定義によります。

実際には、暗騒音に埋もれる前に尾が 60 dB ぶんきれいに減衰することはまれなので、RT60 はほとんどの場合「外挿」で求めます。よりきれいで小さい範囲の減衰の傾きを測り、それを 60 dB へ換算します。

| 指標 | 測る減衰範囲 | 換算 |
|------|--------------|------|
| T20 | −5 dB 〜 −25 dB | ×3 → 60 dB |
| T30 | −5 dB 〜 −35 dB | ×2 → 60 dB |
| EDT | 0 dB 〜 −10 dB | ×6 → 60 dB |

最初の数 dB を飛ばすのは、減衰の出だしが直接音と最も強い初期反射に支配され、まだ「残響」とは言えないためです。

## EDT: 耳が実際に聞いているもの

**初期減衰時間（EDT）** は、減衰の最初の 10 dB だけを測り、その傾きを 60 dB へ換算します。尾の出だし、つまり空間の生き生きとした響きを耳が判断する部分に重みを置きます。

理想的で完全に拡散した部屋では EDT と RT60 は一致します。現実の部屋では食い違います。

- **EDT が RT60 より大きく短い** — 強い初期反射が減衰の出だしを急にするため、長い後部残響が示すよりも部屋がタイトに聞こえます。近接面が反射的で遠い面が吸音的な部屋でよく見られます。
- **EDT が RT60 に近い** — なめらかでよく拡散した減衰で、体感の残響感が測定値と一致します。

知覚は EDT に追従するため、RT60 が同じでも 2 つの部屋はかなり違って感じられます。EDT は「残響感」のよい予測子、RT60 は持続音が次の音にどれだけにじむかのよい予測子です。

## 数値の読み方

| RT60 | 印象 |
|------|------|
| < 0.3 s | デッド — ブース、処理されたコントロールルーム |
| 0.3〜0.6 s | タイト — 寝室、小規模スタジオ |
| 0.6〜1.2 s | ライブ — 録音用の部屋、広いラウンジ |
| 1.2〜2.5 s | 残響豊か — コンサートホール |
| > 2.5 s | 洞窟的 — 大聖堂、広い石造りの空間 |

RT60 は**容積**が大きいほど長くなり、**吸音**が多いほど短くなります。大きく硬い面の空間は長く鳴り、小さく柔らかい家具のある部屋は鳴りません。この関係こそが、libsonare が減衰だけから等価な部屋を推定できる理由です。

::: details libsonare が減衰時間をどう測るか
libsonare はエネルギー減衰曲線（インパルス応答の二乗のシュレーダー逆積分）から残響時間を導きます。T20／T30 領域で減衰に直線を当てはめ、曲線が十分に直線的でない区間を棄却したうえで、傾きを 60 dB へ外挿します。EDT は最初の 10 dB に当てはめます。帯域別の残響時間は、オクターブ帯域でフィルタした応答に同じ当てはめを行って算出し、これが帯域別減衰の表になります。入力がクリーンなインパルスではなく通常の音楽の場合、減衰は信号の隙間やノートの切れ目からブラインドで復元され、信頼度スコアはその復元された減衰がどれだけきれいだったかを反映します。
:::
